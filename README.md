# LLM 기반 업무 자동화 프롬프트 패키지

## 프로젝트 목적

연구보고서 개요 자동 생성을 위한 LLM 기반 프롬프트를 설계하고,

- 모델 비교
- 프롬프트 개선
- 환각(Hallucination) 검증
- 장기 문맥 유지

를 수행하였다.
