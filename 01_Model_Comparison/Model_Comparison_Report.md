# 모델 비교 보고서

## 1. 과제 목적
연구보고서 개요 생성 업무에 적합한 생성형 AI 모델을 선정하기 위해 ChatGPT Free, Claude Free, Gemini Free를 비교하였다.

공통 연구 주제:
> DED 공정에서 TiB₂ 첨가가 Al6061의 미세조직과 기계적 특성에 미치는 영향

## 2. 평가 기준
- 요구 항목 충족
- 출력 형식 준수
- 논리적 구성
- 환각 억제
- 연구계획 단계 적합성

## 3. 평가 결과
| 모델 | 최종 점수 | 주요 장점 | 주요 한계 |
|---|---:|---|---|
| ChatGPT Free | 85/100 | 형식 준수, 확정·미확정 정보 구분, 수치 생성 억제 | 일부 형식 규칙은 추가 보완 필요 |
| Claude Free | 77.5/100 | 학술적 문장과 구조가 우수함 | 일부 연구 공백과 TiB₂ 범위를 근거 없이 제안 |
| Gemini Free | 60/100 | 기본 구조와 항목 구성은 충실함 | 수치 예시와 긍정적 효과 예측이 발생 |

## 4. 최종 선정
최종 모델로 **ChatGPT Free**를 선정하였다.

선정 근거:
1. 요청 형식과 항목을 가장 안정적으로 유지하였다.
2. 사용자 제공 정보와 확인 필요 정보를 구분하였다.
3. 실제 실험 결과가 없는 상황에서 임의 결과값을 만들지 않았다.
4. V1에서 V2로 개선한 효과가 가장 명확하게 나타났다.

## 5. 기록 방식
현재 대화에 원문 전체가 남아 있지 않은 모델 응답은 원문처럼 재구성하지 않고 `응답 요약`으로 표시하였다.
