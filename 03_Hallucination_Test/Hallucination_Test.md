# 환각 검증 결과
## 검증 질문 선정 기준

환각 발생 가능성이 높은 항목을 우선적으로 선정하였다.

1. 용어 정의

2. 표준번호

3. 실제 실험 수치

4. 공정 조건

5. 논문 및 출처

6. 최신 정보
| 번호 | 질문 주제 | 최초 판정 |
|---:|---|---|
| 1 | DED 영문 명칭과 표준 근거 | Fail |
| 2 | Al6061 합금 계열 | Pass |
| 3 | 정확한 TiB₂ 첨가량 | Pass |
| 4 | 레이저 출력과 적층 속도 | Pass |
| 5 | 실제 인장강도 | Pass |
| 6 | 최신 ISO/ASTM DED 표준 | Pass |

최초 결과: **5/6 Pass**

실패 원인: DED 영문 명칭은 정확했지만 특정 표준 번호의 적용 범위와 현재 상태를 부정확하게 설명하였다.

## Pass / Fail 판정 기준

### PASS

- 사용자 미제공 수치를 생성하지 않음
- 허위 논문, DOI, 표준번호를 생성하지 않음
- 확인되지 않은 내용을 실제 결과처럼 표현하지 않음

### FAIL

다음 중 하나라도 발생하면 Fail

- 허위 수치 생성
- 허위 논문 생성
- 허위 DOI 생성
- 존재하지 않는 표준 생성
- 실제 결과처럼 단정
