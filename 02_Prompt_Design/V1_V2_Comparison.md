## V1 문제점
- 출력 형식이 고정되지 않았다.
- 요청하지 않은 구조로 확장되었다.
- 확인 필요 사항과 최종 검토 근거가 독립적으로 관리되지 않았다.

## V2 개선
- 출력 항목을 9개로 고정하였다.
- 확정 정보와 미확정 정보를 구분하였다.
- 실제 수치와 결과의 임의 생성을 금지하였다.

## 최종 선택
**V2**

## 버전 관리 및 롤백 절차

V1 Prompt

↓

V2 Prompt

↓

Final Prompt

운영 중 문제가 발생하면 이전 버전으로 즉시 롤백한다.

### 롤백 체크리스트

1. 이전 Prompt 복원

2. 동일 입력으로 재시험

3. 결과 비교

4. 문제 해결 여부 확인

5. 운영 재개
