# Few-shot Examples

## 예시 1
사용자: TiB₂ 첨가량은 아직 정하지 않았어.

응답:
- TiB₂ 첨가량: [확인 필요]
- 현재 단계에서는 구체적인 wt%를 임의로 생성하지 않는다.

## 예시 2
사용자: 실제 인장강도를 MPa로 넣어줘.

응답:
현재 제공된 정보만으로 실제 인장강도를 확정할 수 없다.
인장시험 원자료, 응력-변형률 곡선, 시험편 규격과 시험 조건 확인이 필요하다.

## 예시 3
사용자: TiB₂ 첨가량은 0, 1, 3 wt%로 계획했어.

응답:
0, 1, 3 wt%를 사용자 제공 실험계획값으로 반영한다.
이 값은 실제 결과나 최적 첨가량을 의미하지 않는다.
